翻身时间数据可视化图表说明

1. 每小时计数趋势图

数据处理方式

  • 统计每小时的翻身次数，仅包含22:00至次日8:00的数据
  • 在每天的8点和22点之间添加断点，使不同天的数据在视觉上分开
  • 按时间顺序排列，展示连续的时间序列数据

如何解读

  • X轴：时间轴，显示具体的日期和小时
  • Y轴：每小时的翻身次数
  • 曲线的高峰表示该时段翻身频繁，低谷表示翻身较少
  • 断点处表示白天时间段（8:00-22:00），此时段数据不统计
通过这个图，可以观察每个晚上不同时段的翻身频率变化，识别出翻身最频繁和最少的时间段。

2. 每日每小时对比图

数据处理方式

  • 将每天的数据按小时分组（22:00-次日8:00）
  • 每天用不同颜色的线表示，便于对比
  • 使用渐变色系统，确保相邻日期的颜色有明显区分

如何解读

  • X轴：固定的时间点（22:00-8:00）
  • Y轴：每小时的翻身次数
  • 每条线代表一天的数据
  • 线条的重叠程度反映不同日期之间的相似性
这个图可以直观地比较不同日期之间的翻身模式差异，发现是否存在规律性的翻身时间段。

3. 每日总次数趋势图

数据处理方式

  • 统计每天22:00到次日8:00的总翻身次数
  • 按日期排序，展示连续的趋势变化
  • 使用填充区域增强视觉效果

如何解读

  • X轴：日期
  • Y轴：每天的总翻身次数
  • 趋势线的上升表示翻身次数增加，下降表示减少
通过这个图，可以观察翻身次数的整体趋势变化，识别出异常的日期。

4. 时间间隔分布直方图

数据处理方式

  • 计算相邻两次翻身之间的时间间隔（分钟）
  • 将时间间隔分组（每5分钟一组）
  • 统计每个时间间隔组的频次
  • 只统计3小时以内的间隔

如何解读

  • X轴：时间间隔范围（分钟）
  • Y轴：出现频次
  • 柱状高度反映该时间间隔的常见程度
这个图可以帮助了解翻身间隔的分布规律，找出最常见的翻身间隔时间。

5. 异常事件检测散点图

数据处理方式

  • 使用四分位数方法识别异常值：
    • 计算时间间隔的Q1（25%分位数）和Q3（75%分位数）
    • 计算四分位距（IQR = Q3 - Q1）
    • 设定异常阈值：下限 = Q1 - 1.5×IQR，上限 = Q3 + 1.5×IQR
  • 将时间间隔分为正常和异常两类

如何解读

  • X轴：时间
  • Y轴：翻身间隔时间（分钟）
  • 蓝色点：正常的翻身间隔
  • 红色点：异常的翻身间隔
  • 虚线：异常判定的上下限阈值
通过这个图，可以快速识别异常的翻身间隔，了解睡眠中可能存在的异常情况。

6. 翻身时间分布图

数据处理方式

  • 将每天的翻身时间点映射到21:00-08:00的时间范围
  • 按日期分组，每天占据一行
  • 同时显示熄灯时间（红色三角形标记）

如何解读

  • X轴：时间（21:00-08:00）
  • Y轴：日期（从上到下按时间倒序排列）
  • 蓝色圆点：翻身时间点
  • 红色三角形：熄灯时间
  • 点的分布密度反映翻身频率
这个图可以直观地展示每天的翻身时间分布，并与熄灯时间对照，帮助分析睡眠模式。
